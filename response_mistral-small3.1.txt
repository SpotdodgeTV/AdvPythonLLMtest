User: Write a python script that takes a paragraph of text and returns, all words greater than three letters long in alphabetical order, and the number of times the word “and” appears in the text and prints the results to console.
Response: Certainly! Below is a Python script that takes a paragraph of text, extracts all words greater than three letters long, sorts them alphabetically, counts the number of times the word "and" appears, and prints the results to the console.

```python
import re
from collections import Counter

def process_text(paragraph):
    # Use regex to find words
    words = re.findall(r'\b\w+\b', paragraph)

    # Filter words greater than three letters long and convert to lowercase
    filtered_words = [word.lower() for word in words if len(word) > 3]

    # Sort the filtered words alphabetically
    sorted_words = sorted(filtered_words)

    # Count the occurrences of the word "and"
    word_counts = Counter(words)
    and_count = word_counts['and']

    # Print the results
    print("Words greater than three letters long (alphabetical order):")
    print(sorted_words)
    print(f"\nThe word 'and' appears {and_count} times.")

# Example usage
paragraph = """
Python is a versatile and powerful programming language. It is widely used for web development, data analysis,
and machine learning. Many developers appreciate its simplicity and readability.
"""

process_text(paragraph)
```

### Explanation:
1. **Regular Expression**: The `re.findall(r'\b\w+\b', paragraph)` function call uses a regular expression to find all words in the paragraph. The `\b\w+\b` pattern matches word boundaries.
2. **Filtering**: The list comprehension `[word.lower() for word in words if len(word) > 3]` filters out words that are shorter than or equal to three letters and converts the remaining words to lowercase.
3. **Sorting**: The `sorted(filtered_words)` function sorts the filtered words alphabetically.
4. **Counting**: The `Counter(words)` function from the `collections` module counts the occurrences of each word in the paragraph. The `and_count` variable stores the count of the word "and".
5. **Printing**: The script prints the sorted list of words and the count of the word "and".

You can replace the `paragraph` variable with any text you want to process.
